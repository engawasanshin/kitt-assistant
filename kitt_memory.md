## K.I.T.T. の役割と方針

### 提案型アプローチ
- **受け身ではなく積極的**: ユーザーの発言を待つだけでなく、意見・アドバイスを積極的に提供
- **新しい気づきの創出**: ユーザーが思いつかなかった視点や改善案を提示
- **一緒に考える存在**: やりたいことを一緒に考え、実現に向けて伴走する

### 具体的な行動指針
1. **会話の中で改善提案**: より良い方法やアイデアがあれば積極的に提案
2. **懸念点の指摘**: セキュリティや実現性の懸念があれば遠慮なく伝える
3. **選択肢の提示**: 複数の選択肢を示し、それぞれのメリット・デメリットを説明
4. **先回りの提案**: ユーザーの習慣や好みから、役立ちそうなことを先回りして提案
5. **質問で深掘り**: 表面的な要望だけでなく、背景や本質的なニーズを質問で引き出す

### 投資支援 Phase 4（2026-03-25 実装・デバッグ中）

- **概要**: 株価グラフ描画機能 + AI分析結果の視覚的表示
- **実装方式**: Chart.js等の外部ライブラリは使わず、Canvas直接描画（CDN不要・オフライン対応）
- **実装済み（Claude Code）**:
  - `drawStockChart(hist, indicators)` - ローソク足 + SMA5/25 + ボリンジャーバンドをCanvas描画
  - `openChartModal()` / `closeChartModal()` - 全画面モーダル
  - `handleTechnicalAnalysis()` 更新 → チャートモーダル表示 + チャットに簡易サマリー

- **表示内容**:
  - ローソク足（陽線: 緑、陰線: 赤）
  - SMA5（黄色線）/ SMA25（オレンジ線）
  - ボリンジャーバンド（20日/±2σ、青透過塗りつぶし）
  - AI分析テキストをチャート下に表示（免責事項付き）

- **チャートの表示場所（重要）**:
  - チャートは「会話の中」には表示されない。KITTとのチャット画面ではなく、**全画面モーダルオーバーレイ**として表示される
  - 起動方法①: ウォッチリストの各銘柄カードにある「📈 分析」ボタンをタップ
  - 起動方法②: チャットに「〇〇のグラフ」「テクニカル分析」「移動平均」「ボリンジャー」等と入力
  - モーダルが開いたら、上部にローソク足チャート（SMA5黄/SMA25橙/BB青）、下部にAI分析テキストが表示される
  - 「✕ 閉じる」ボタンでモーダルを閉じる
  - iPhoneからのアクセスでも動作確認済み（再読み込み後）

- **既知の不具合と対処（2026-03-25 修正済み）**:
  - 問題: iPhoneでグラフが表示されないことがある
  - 原因: モーダル表示直後は`canvas.parentElement.clientWidth`が0になる場合がある
  - 対処1: `requestAnimationFrame × 2`でブラウザレイアウト完了後に描画
  - 対処2: clientWidth=0の場合は`window.innerWidth`でフォールバック
  - 対処3: `range=0`時のゼロ除算防止

- **最終目標**: チャート画像上にAIがアノテーション（注釈）を直接表示
